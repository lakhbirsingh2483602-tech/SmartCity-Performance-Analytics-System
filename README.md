# 🏙️ Smart City Performance Analytics System

## 📌 Overview

The **Smart City Performance Analytics System** is an end-to-end data analytics project that transforms raw urban data into meaningful business insights. The project focuses on analyzing multiple smart city datasets, including air quality, traffic, water supply, electricity consumption, waste management, and public complaints.

Using **Python**, **MySQL**, **SQL**, and **Power BI**, the project demonstrates the complete analytics workflow—from data loading and cleaning to exploratory data analysis (EDA), database querying, dashboard creation, and reporting. The final insights are presented through an interactive Power BI dashboard, a detailed analytical report, and a professional presentation created using Gamma.

---

## 📂 Dataset

The project uses multiple smart city datasets containing historical records related to:

* 🌍 Air Quality (AQI, PM2.5, PM10)
* 🚗 Traffic and Vehicle Data
* 💧 Water Supply
* ⚡ Electricity Consumption
* ♻️ Waste Management
* 📢 Public Complaints

The datasets are stored in CSV format and processed before analysis.

## 📈 Project Workflow

### 1. Data Loading

* Imported multiple CSV datasets using Python.
* Verified data types and dataset structure.

### 2. Exploratory Data Analysis (EDA)

* Examined data distributions and summary statistics.
* Identified missing values, duplicates, and outliers.
* Explored trends and relationships between variables.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Standardized column names and formats.
* Converted date and time fields.
* Prepared clean datasets for analysis.

### 4. SQL Analysis (MySQL)

* Imported cleaned datasets into MySQL.
* Executed SQL queries to analyze:

  * District-wise performance
  * Resource consumption
  * Traffic patterns
  * Complaint statistics
  * Air quality trends

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* KPI Cards
* District-wise Analysis
* Traffic Analytics
* Air Quality Monitoring
* Water Supply Insights
* Electricity Consumption
* Waste Management Analysis
* Public Complaint Analytics
* Interactive Filters and Slicers

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/yourusername/smart-city-performance-analytics.git
```

### Install required Python libraries

```bash
pip install pandas numpy matplotlib seaborn mysql-connector-python
```

### Run the Python scripts

```bash
python data_cleaning.py
```

### Import cleaned data into MySQL

* Create a MySQL database.
* Import the cleaned CSV files.
* Execute the provided SQL scripts.

### Open Power BI Dashboard

* Open the `.pbix` file using Power BI Desktop.
* Refresh the data if required.
